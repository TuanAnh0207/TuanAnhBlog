async/await

Viết code “đồng bộ” cho luồng bất đồng bộ; luôn bọc try/catch khi cần.

// Demo fetch với async/await
async function fetchJson(url){
  const res = await fetch(url);
  if(!res.ok) throw new Error(res.status+':'+res.statusText);
  return await res.json();
}

(async () => {
  try{
    const data = await fetchJson('https://api.github.com');
    console.log(data);
  }catch(e){ console.error('Fetch error', e); }
})();

Retry & timeout

function timeout(ms){ return new Promise((_,r)=> setTimeout(()=> r(new Error('Timeout')), ms)); }
await Promise.race([ fetchJson(url), timeout(3000) ]);

FAQ

Vì sao await trong vòng lặp for…of chậm? Vì từng lần đợi xong mới lặp tiếp. Dùng Promise.all để song song khi độc lập.

Patterns cho Promise

  • Queue: giới hạn số request đồng thời.
  • Memoization: cache promise theo key để tránh fetch trùng.
  • Circuit breaker: tạm ngưng gọi khi tỉ lệ lỗi cao.
