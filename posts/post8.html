Event delegation

Gắn listener lên container để xử lý nhiều phần tử con động.

Hiệu suất

Giảm reflow bằng batch cập nhật, dùng DocumentFragment, và classList thay inline style.

// Event delegation demo
const list = document.querySelector('#todos');
list.addEventListener('click', (e) => {
  const item = e.target.closest('li');
  if(!item) return;
  item.classList.toggle('done');
});
